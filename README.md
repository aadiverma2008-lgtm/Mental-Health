# Mental Health Awareness & Suicide Prevention Agent

An Agentic AI-powered solution for mental health support, early detection, and crisis prevention.

## 🎯 Features

1. **Awareness & Education** - Accessible resources on mindfulness, coping strategies, and self-care
2. **Early Detection (RAG)** - Retrieval-Augmented Generation to identify distress signals
3. **Empathetic Support** - Conversational AI agent with active listening and empathy
4. **Risk Prediction** - Forecast stress peaks and suggest preventive strategies
5. **Human Support Connection** - Direct links to helplines, counselors, and community support

## 🏗️ Architecture

```
mental-health/
├── backend/          # FastAPI backend with AI agents
├── frontend/         # React-based user interface
├── models/           # ML models and RAG components
├── data/             # Training data and knowledge base
└── docs/             # Documentation
```

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Node.js 16+
- OpenAI API key or similar LLM access

### Installation

1. Clone the repository
2. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
3. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Running the Application

1. Start the backend:
   ```bash
   cd backend
   python main.py
   ```

2. Start the frontend:
   ```bash
   cd frontend
   npm start
   ```
